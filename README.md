DocTalk AI

Chat with your documents using the power of Gemini AI

DocTalk AI lets you upload any document or image and have a real conversation with it. Ask questions, get summaries, extract key info — all powered by Google Gemini AI.

 Features

 Upload documents (PDF, images) and chat with them instantly
 Powered by Google Gemini AI for intelligent, context-aware responses
 Real-time chat interface with smooth conversation flow
 Secure file handling via Firebase Storage
 Fast and lightweight — built with React + Vite
 Responsive design, works on desktop and mobile
